# Data Analysis Project

## Introduction
This project aims to analyze customer preferences data for WE Telecom. It involves several steps, starting from raw data processing to building predictive models.

## Project Structure

- **app/**: Contains the `app.py` file, which serves as the entry point to run the application.
- **assets/**: Contains assets such as images or other files used in the project.
- **Data/**: Contains the data used in the project, divided into `clean` and `raw` folders.
  - **clean/**: Contains the cleaned data.
  - **raw/**: Contains the raw data.
- **metadata/**: Contains metadata information about the data.
- **models/**: Contains the trained models.
- **noteBook/**: Contains Jupyter notebooks used for data analysis.
- **src/**: Contains programming files used in the project.

## Requirements
To ensure the project runs correctly, make sure to install the following packages:

```bash
pip install -r requirements.txt
```

## How to Use
1. Run the `app.py` file:
   ```bash
   python app/app.py
   ```
2. Follow the instructions displayed on the screen.
